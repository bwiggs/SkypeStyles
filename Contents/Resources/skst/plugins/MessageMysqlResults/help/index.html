To beautify SQL results just copy paste the MySQL console result into the chat begining from "+------+--------+---[...]" part.
It'll transform it into an array grid which is much more readable than the raw output.

Example: 

+-----------+-------------+------------------+------------+------------+
| region_id | region_code | region           | latitude   | longitude  |
+-----------+-------------+------------------+------------+------------+
|         1 | Afr         | Africa           |   0.460197 |  19.237293 |
|         2 | Ant         | Antarctic Region | -77.830002 | 166.669998 |
+-----------+-------------+------------------+------------+------------+
2 rows in set (0.00 sec)
