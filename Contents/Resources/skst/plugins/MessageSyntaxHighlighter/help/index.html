To highlight code, simply surround it with [language][/language] markups, replacing language with the corresponding language abbriv/code.
You don't even need the closing tag if the end of the code is the end of your message.

Example: 

[js]
if ( a == 10 )
{
	alert('Hello World!');
}
[/js]
